# [USACO2001]牛的骨牌

## Description

奶牛们正在玩一种包含$N$个骨牌的游戏.每个骨牌的两端各有1个0到9 (包含0和9)的数字, 例如：
下面三块排成一排的骨牌表示2个10进制数:
5 3 4<br>2 4 1
分别表示了534和241.当然了，每块骨牌都可以旋转180度，也就是交换两端的数字，例如:
5 2<br>2 —&gt; 5
这种特殊的游戏的目标就是使得$N$块骨牌所组成的2个10进制数之和越大越好.比如上面的例子，最大的和就是775. 你的任务就是计算出这个最大值.


## Sample Input 1

```
3
1  4
2  5
34
```

## Sample Output 1

```
775
```
